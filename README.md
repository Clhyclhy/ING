# AstrBot 目标整理助手

将用户目标任务按两级标题整理：

- 一级：任务合集（1、2、3）
- 二级：合集内任务（①②③）

## 功能

- 新建任务合集（可自定义合集名）
- 向合集添加任务（标题 + 说明，说明可为空）
- 查看两级标题结构
- 输入编号查看任务详情（如 1.1）
- 可选纯文本快捷查询（查看任务 / 1.1）

## 指令

- /任务 新建 <合集名>
- /任务 添加 <合集序号> <任务标题> | <任务说明(可空)>
- /任务 查看
- /任务 详情 <合集序号.任务序号>
- /任务 帮助

当查看某条任务详情且说明为空时，插件会追问是否要补充说明；若回复“是”，下一条消息将被作为任务说明写入。

## 示例

- /任务 新建 学习计划
- /任务 添加 1 学完第一章 | 完成课后题并整理笔记
- /任务 查看
- /任务 详情 1.1
